# Hyper-V Telegram Control Bot
Легковесный и удобный Telegram-бот на Python для удаленного управления виртуальными машинами в диспетчере Hyper-V через PowerShell. Идеально подходит для системных администраторов и разработчиков, которым нужен быстрый доступ к своим ВМ с мобильного телефона или другого устройства без необходимости заходить по RDP.

🚀 Основные возможности
Интерактивный список ВМ: Получение подробных карточек для каждой виртуальной машины (Имя, Статус, Выделенная память, Количество vCPU, Аптайм).

Управление в один клик: Каждая карточка ВМ оснащена инлайн-кнопками для быстрого выполнения базовых действий:

▶️ Включение (Start-VM)

⏹️ Корректное завершение работы (Stop-VM)

🔄 Принудительная перезагрузка (Restart-VM -Force)

Мониторинг запущенных ВМ: Быстрый вывод списка всех активных машин с их текущей нагрузкой.

Безопасность: Встроенная проверка по Telegram ID (ADMIN_ID) — бот будет реагировать на команды только от владельца.

🛠️ Требования к системе
Для корректной работы скрипта необходимы:

Операционная система: Windows 10 / 11 Pro/Enterprise или Windows Server с установленным и настроенным компонентом Hyper-V.

Python: Версия 3.8 или выше.

Права администратора: Бот выполняет командлеты PowerShell, требующие повышенных привилегий в системе.

📦 Быстрый старт и установка
Шаг 1. Клонирование репозитория
Bash
git clone https://github.com/ВАШ_НИКНЕЙМ/название-репозитория.git
cd название-репозитория
Шаг 2. Установка зависимостей
Проект использует библиотеку pyTelegramBotAPI (telebot). Установите её через pip:

Bash
pip install pytelegrambotapi
Шаг 3. Настройка конфигурации
Откройте файл vm_bot.py и заполните конфигурационные переменные в начале файла:

Python
# --- НАСТРОЙКИ ---
BOT_TOKEN = "ВАШ_ТОКЕН_БОТА"      # Токен, полученный от @BotFather
ADMIN_ID = 123456789              # Ваш числовой Telegram ID (можно узнать у @userinfobot)
Шаг 4. Запуск бота
⚠️ Важно: Терминал (CMD или PowerShell) необходимо запустить от имени Администратора, иначе Windows заблокирует доступ к командам Hyper-V.

DOS
python vm_bot.py
🕹️ Использование
Перейдите в чат с вашим созданным ботом в Telegram и отправьте команду /start.

Если ваш Telegram ID совпадает с ADMIN_ID, бот отправит приветственное сообщение и откроет главное меню.

Используйте кнопки на клавиатуре для навигации:

📋 Список ВМ Hyper-V — генерирует отдельные карточки управления для каждой машины.

📊 Запущенные ВМ — показывает текстовую таблицу только с работающими в данный момент системами.

🔒 Безопасность и ограничения
Скрипт использует кодировку cp1251 для корректного чтения вывода PowerShell в русскоязычных версиях Windows, предотвращая появление нечитаемых символов («кракозябр») при ошибках.

Все команды PowerShell выполняются через стандартный модуль subprocess с флагом check=True для отлова системных сбоев.
